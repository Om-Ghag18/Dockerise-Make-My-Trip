# Dockerise-Make-My-Trip

✈️ Travel Booking App

A full-stack Travel Booking Application built using Node.js, Express, React, and MongoDB, containerized with Docker and orchestrated using Docker Compose.This application allows users to register, view travel options, and create bookings through a simple web interface.


🏗️ Architecture

* The project follows a 3-tier architecture:

* Frontend – React application
* Backend – Node.js + Express REST API
* Database – MongoDB
All services run inside Docker containers.
